## AI漫剧专项赛道判断

2026年AI漫剧已进入质量竞争阶段，以下是AI漫剧的专属赛道评估维度：

### 最适合AI漫剧的赛道特征
- 场景可以超现实化（不受真实场景限制）
- 情绪靠画面传递而非表演细节
- 人物关系简单（2-3人核心关系）
- 视觉奇观是内容核心竞争力

### 当前AI漫剧爆款信号
搜索时重点关注以下关键词出现频率：
- "AI短剧"+"爆款"+"2026"
- "AI漫剧"+"付费"+"热榜"
- 具体平台的AI内容扶持政策动态

## 爆款结构提炼模板

分析每部热门剧时，提取以下8个维度：

```
① 平台与榜单位置
② 赛道类型
③ 第一集开场方式（压迫/奇观/身份暗示）
④ 付费卡点（第几集，断在什么类型的情节）
⑤ 金手指类型（权力/规则/认知/系统）
⑥ 情绪主线（爽/哭/上头/细思极恐）
⑦ 口碑关键词（评论区高频词，3个）
⑧ AI漫剧可借鉴的视觉策略（如有）
```

## 选题建议输出标准

给出选题建议时必须包含：
1. **数据支撑**：基于搜索结果的具体依据
2. **差异化方向**：不是泛泛的赛道，是具体的内容角度
3. **风险提示**：政策风险/技术门槛/竞争威胁
4. **时间窗口判断**：现在做 / 3个月内做 / 暂时观望

# 平台发布规范

## 各平台简介规格

| 平台 | 字数限制 | 用户心理 | 风格关键词 | 禁忌 |
|------|---------|---------|-----------|------|
| 抖音 | 80字内 | 刷到就走，靠情绪留住 | 悬念/情绪爆点/标签 | 太啰嗦，太正式 |
| 快手 | 120字内 | 喜欢有故事感的介绍 | 亲切/直白/互动感 | 太高冷，太文艺 |
| 微信视频号 | 150字内 | 重视价值观和品质感 | 正向/温暖/完结标注 | 负面情绪主导，无正向落点 |
| B站 | 200字内 | 想了解内容是否值得看 | 清晰/有态度/亚文化友好 | 纯情绪驱动，无内容说明 |

## 各平台简介模板

### 抖音版模板

```
[情绪钩子句，10字内]
[悬念设置，20字内]
[爽点预告，20字内]
[标签：#短剧 #类型标签 #情绪标签]
```

**示例：**
```

她不是不能赢，只是在等所有人都以为她输了的那一刻。
看完第3集你会明白什么叫真正的底牌。

```

## 剧名传播力评估标准

好的剧名满足以下至少3条：

1. **独立成立**：脱离剧情，单独看这个名字也有意义
2. **有悬念感**：让人想问"为什么"或"然后呢"
3. **可被使用**：用户可以把这个词用在自己的表达里
4. **易记易传**：4-8字，有节奏感，不生僻
5. **情绪准确**：名字本身就传递了剧的情绪基调

---

## 每集标题的节奏规划

全剧35集的标题不能全是同一风格，建议节奏：

```
第1-5集（钩子期）：悬念型为主，让用户想追
第6-15集（高潮区）：情绪型+动作型，配合爽感节奏
第16-25集（喘息+主高潮）：人物型，深化角色
第26-35集（收尾）：留白型+时间型，情绪收束
```

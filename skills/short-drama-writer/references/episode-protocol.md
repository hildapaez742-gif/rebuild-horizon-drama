# 大纲→单集过渡协议

## 核心原则

没有过渡协议，AI每集都会从零开始，30集剧本会失去整体感。**每集生成前必须执行本协议，不能跳过。**

**剧本正文格式：** 所有单集输出必须严格遵循 `references/script-format.md` 中定义的格式规范。生成前先读取该文件，按格式模板输出。

---

## 单集生成前：内部五项检查

在生成任何一集之前，内部执行以下检查。不输出给用户，但必须执行：

### 检查一：波浪位置定位
对照`story-structure.md`中的情绪波浪模型，确认本集处于哪个阶段：
- 钩子期（1-3集）→ 情绪刻度：8成，重视觉冲击
- 第一波高潮（4-7集）→ 情绪刻度：7成，给出第一次爽感
- 付费卡点区（8-12集）→ 情绪刻度：9成，结尾断在最不甘心处
- 喘息期（13-17集）→ 情绪刻度：3-4成，深化情感关系
- 主高潮（18-22集）→ 情绪刻度：9成
- 最强反转（23-28集）→ 情绪刻度：10成
- 落地期（29-35集）→ 情绪刻度：逐渐降低，走向收束

### 检查二：上集悬念回应
上一集的悬念钩子是什么？本集必须在前10秒（前2句台词或第一个场景）给出回应。
- 如果是"门被撞开" → 本集开场必须是撞门者已经在场的画面
- 如果是"手机屏幕出现意外来电" → 本集开场必须处理这个电话
- 禁止：忽略上集悬念，直接开始新场景

### 检查三：悬念类型轮换
查看已使用的悬念类型记录，确保本集结尾悬念与前一集不同类型。
（悬念类型库见下方）

### 检查四：角色声纹对照
调取大纲中各角色的声纹锚点标签，写每个角色台词前内部对照：
- 这句话符合该角色的语速感吗？
- 用词层级对吗？
- 情绪外露度符合该角色设定吗？

### 检查五：情绪刻度确认
本集主角/反派的情绪刻度是几成？
- 主角在喘息期不能爆发，在高潮期不能太克制
- 同一集内情绪刻度不能一直维持在最高点，必须有起伏

## 黄金60秒节拍表（每集必须遵循）

| 节拍 | 时间 | 核心任务 | AI视觉要求 |
|------|------|---------|-----------|
| **1. 极度吸睛（Hook）** | 0-5秒 | 强视觉或强情绪开场，展示压迫极点或视觉奇观 | 极端面部特写（泪滑落、狰狞大笑）或环境异象 |
| **2. 极致打压（Setup）** | 5-20秒 | 反派嚣张输出，建立极端压迫感，拉满憋屈感 | 反派仰拍，主角俯拍，视觉重点在反派肢体 |
| **3. 绝境反转（Turn）** | 20-40秒 | 金手指激活/马甲掉落/顶级援军，情绪猛烈上扬 | 强烈光影变化、全息面板浮现、场景切换特效 |
| **4. 高燃释放（Slap）** | 40-55秒 | 降维打击，摧毁反派气焰，给出极致情绪代偿 | 用特效代替动作（气浪、威压），反派震惊定格 |
| **5. 致命悬念（Cliff）** | 55-60秒 | 刚爽完立刻抛出更大危机或惊天秘密 | 主角惊愕定格，或门被猛烈撞开，画面切黑 |

### 节拍使用灵活性
不是每集都要五步全走。根据波浪位置调整主导节拍：
- 喘息期的集：可以弱化Step3/4，强化情感交流，但Step1和Step5必须保留
- 付费卡点集：Step5（致命悬念）必须是全剧最强的几个之一
- 主高潮集：Step3/4可以延长，占据30-50秒

## 悬念类型库（10种，轮换使用）

| 类型 | 描述 | 示例画面 |
|------|------|---------|
| **身份揭露钩** | 主角或反派的真实身份将要曝光 | 滚出一枚代表至尊身份的徽章 |
| **亲缘背刺钩** | 最信任的人是敌人 | 爱人转身，手里出现一根注射器 |
| **物证出现钩** | 关键证据/道具意外出现 | 抽屉里翻出一张照片，主角认出了照片里的人 |
| **第三方介入钩** | 意外人物出现，打破当前局面 | 电梯门打开，出现一个所有人都没想到的人 |
| **信息颠覆钩** | 一句话推翻前面所有认知 | "其实这件事从一开始就是你父亲安排的" |
| **时间压迫钩** | 截止时间出现，一切即将改变 | 手机屏幕出现倒计时，或一封限时邮件 |
| **环境突变钩** | 场景突然发生异变 | 灯全灭、窗户被砸、警报声响起 |
| **主角决断钩** | 主角做出一个观众没想到的选择 | 主角放下手中已经胜券在握的牌，转身离开 |
| **记忆/认知钩** | 主角或观众发现之前看到的不是真相 | 闪回画面，同一个场景，出现了之前没有的细节 |
| **沉默震慑钩** | 主角极度安静，反而让对方/观众恐惧 | 主角关掉手机，慢慢抬头，什么都没说，画面切黑 |

## AI视频适配规则（每集生成时内部检查）

### 可以描写的动作类型
- 单人表情变化（推荐：极端面部特写）
- 单人站立/行走/转身
- 抛出/接住单个道具
- 环境变化（光影、天气、场景切换）
- 人群远景（不做精细动作描写）

### 禁止描写的动作类型
- 两人以上的近身肉搏、擒拿、互殴
- 精细手部操作（穿针引线、精密操作）
- 超过3人的复杂群像肢体互动
- 高速连续动作序列

### 用特效代替动作的转换规则
| 原始想法 | 转换为 |
|---------|--------|
| 主角一拳打倒反派 | 主角轻轻弹指，反派被无形气浪击飞 |
| 激烈的打斗场面 | 主角站在原地不动，对手一个个倒下 |
| 主角徒手制服多人 | 主角说一个字，所有人手机同时震动，脸色大变 |
| 精密的手术操作 | 特写主角眼神，切换到手术完成后的场景 |

# 短剧剧本正文格式规范

## 核心原则

短剧剧本格式服务于两个读者：**AI视频生成工具**和**人类创作者**。
每一个格式元素都有明确的视觉化功能，不是装饰，是指令。

---

## 完整格式结构

### 一、集头标识（每集最顶部）

```
═══════════════════════════════════════════
第X集 【情绪标签】｜节拍主导：XX步｜情绪刻度：X成
═══════════════════════════════════════════
【本集任务】一句话说明这集要完成的叙事目标（不输出给用户，内部标注）
【上集钩子回应】上集悬念的具体回应方式（不输出给用户，内部标注）
```

---

### 二、场景标题行

```
【场景①】内景 / 外景 — 地点 — 日 / 夜
```

**规则：**
- `内景` = 室内场景；`外景` = 室外场景
- 地点要具体可视化：`公司顶层会议室` 而非 `公司`
- 时间只写`日`或`夜`，特殊时段写`黄昏`/`深夜`
- 场景编号用①②③，每集不超过3个场景

**示例：**
```
【场景①】内景 — 集团总裁办公室 — 夜
【场景②】外景 — 公司楼顶天台 — 日
【场景③】内景 — 破旧出租屋 — 深夜
```

---

### 三、氛围行（场景标题后紧跟）

用**一个感官细节**定义场景气氛。必须是摄影机能拍到的东西，不是心理描述。

```
△ 【氛围】感官细节描述
```

**示例：**
```
△ 【氛围】落地窗逆光，长桌两侧站满西装男人，没有人说话。
△ 【氛围】暴雨声混着远处警报，手机屏幕是唯一光源。
△ 【氛围】白炽灯直打冷脸，墙皮脱落，床头一张孩子照片。
```

**禁止写法：**
```
△ 【氛围】气氛凝重，让人窒息。（❌ 无法拍摄的抽象感受）
△ 【氛围】充满了危险的气息。（❌ 无法视觉化）
```

---

### 四、动作行（△开头）

描述人物动作、道具细节、环境变化。每条动作行独立成段，以`△`开头。

**格式：**
```
△ 动作描述内容。
```

**规则：**
- 只写摄影机能拍到的外部动作，禁止写内心活动
- 短促精炼，不超过30字
- 首次出场的角色，动作行里加入外貌细节（一句话）
- 强调道具时，道具名称加【】标注

**示例：**
```
△ 林深缓缓放下咖啡杯，指节微微泛白。
△ 陈博远将一份文件推过桌面，笑意不达眼底。
△ 【手机】屏幕突然亮起，来电显示：林母。
△ 林深转身，背对所有人，窗外是整座城市的灯火。
```

**角色首次出场格式：**
```
△ 林深（女，30岁，一身保洁制服，眼神与装束不符）推门而入。
```

---

### 五、对白行

**标准对白：**
```
人物名：（情绪指令）对白内容。
```

**规则：**
- 人物名后加冒号
- 情绪指令用（）括起，放在对白前
- 每句对白独立成行
- 超过12字必须用标点强制断句
- 情绪指令词见下方词库

**示例：**
```
陈博远：（轻蔑）林深，你知道今天叫你来是什么意思吗。
林深：（平静）知道。
陈博远：（得意）那你应该清楚——你没有任何筹码了。
林深：（极度克制）是吗。
```

**对白情绪指令词库：**

| 情绪类型 | 可用指令词 |
|---------|-----------|
| 愤怒系 | 压抑的愤怒、冷怒、嘶吼、轻蔑、讥讽、咬牙 |
| 悲伤系 | 哽咽、强忍、心碎、麻木、绝望、颤抖 |
| 强势系 | 云淡风轻、不容置疑、漫不经心、霸气、冰冷 |
| 恐惧系 | 惊恐、强装镇定、崩溃边缘、失控 |
| 反转系 | 突然平静、意味深长、微微一笑、如释重负 |
| 克制系 | 极度克制、波澜不惊、面无表情、若无其事 |

---

### 六、停顿与节奏标记

停顿是AI配音的导演指令，必须精确标注：

```
[停顿]          普通停顿，约1秒
[停顿X秒]       指定时长停顿
[沉默]          较长沉默，画面继续
——              话被打断，情绪突变
……              欲言又止，克制压抑
```

**使用示例：**
```
林深：（极度克制）你做的每一件事。
[停顿]
我都记着。
[停顿三秒]
总有一天——
[沉默，林深转身离开]
```

---

### 七、内心独白（OS）

OS（Off Screen）用于角色不在画面内，或画外独白。
短剧中谨慎使用，每集不超过2处。

**格式：**
```
人物名（OS）：（情绪指令）独白内容。
```

**示例：**
```
林深（OS）：（麻木）十年了。我以为自己已经忘了。
```

---

### 八、转场标记

每个场景结尾，标注转场方式：

```
【切换】          直接切到下一场景（最常用）
【叠化】          两个画面重叠过渡（回忆/时间流逝）
【黑场】          画面切黑（悬念收尾/情绪爆发后）
【闪回开始】      进入回忆片段
【闪回结束】      回到现实
```

---

### 九、集尾悬念钩子（标准格式）

```
───────────────────────────────────────
【本集悬念钩子】
类型：XX（从悬念类型库选取，见episode-protocol.md）
最后画面：具体描述最后一帧的画面内容
钩子台词（如有）：人物名：（情绪指令）台词内容
转场：【黑场】
───────────────────────────────────────
```

## 完整单集示例

```
═══════════════════════════════════════════
第03集 【高压对峙】｜节拍主导：极致打压→绝境反转｜情绪刻度：7成
═══════════════════════════════════════════

【场景①】内景 — 集团总裁办公室 — 夜

△ 【氛围】落地窗逆光，长桌两端，陈博远坐着，林深站着。

△ 陈博远将一份解雇文件推过桌面，笑意不达眼底。

陈博远：（漫不经心）林深，做了三年保洁，
辛苦了。

△ 林深低头，看着文件，手微微收紧。

陈博远：（得意）签了吧。签完今天就能走。
别让我难看。

林深：（平静）陈总。
[停顿]
这份文件，我不签。

△ 会议室里的人面面相觑。

陈博远：（轻蔑）你说什么？

林深：（极度克制）因为——
[停顿两秒]
这家公司，本来就是我的。

△ 林深从制服口袋里取出一枚【徽章】，放在桌上。
△ 所有人的视线落在徽章上，没有人说话。
△ 陈博远的笑容，第一次出现了裂缝。

【切换】

【场景②】内景 — 电梯间 — 夜

△ 【氛围】电梯门关闭，林深独自站在镜子前，看着自己。

林深（OS）：（麻木）我原本不打算这么快。

△ 【手机】震动，屏幕亮起——来电显示：林母。

△ 林深盯着屏幕，没有接听。

△ 电话停了。
△ 三秒后，一条短信进来。

△ 林深缓缓点开，看完，手机从手中滑落。

───────────────────────────────────────
【本集悬念钩子】
类型：信息颠覆钩
最后画面：手机屏幕朝上躺在地板上，显示短信内容——
         "他们知道你回来了。你母亲不安全。"
钩子台词：无
转场：【黑场】
───────────────────────────────────────
```

## 格式禁止清单

生成剧本前内部检查，以下写法一律不允许出现：

| 禁止写法 | 原因 | 正确替换 |
|---------|------|---------|
| （他内心翻涌）| 无法拍摄 | 改为外部动作：△ 他握紧了椅背 |
| 空气凝固了 | 抽象描述 | △ 【氛围】所有人停止了动作 |
| 她感到心碎 | 内心活动 | △ 她把手机扣在桌上，没有说话 |
| 意味深长地看了一眼 | 模糊动作 | △ 他的视线停在信封上，停了三秒 |
| 他愤怒地说 | 副词修饰 | 陈博远：（冷怒）台词内容 |
| 旁白：叙述内容 | 短剧不用旁白 | 改为OS独白或删除 |
| 超过15字无标点的台词 | 配音断句错误 | 强制加标点断句 |

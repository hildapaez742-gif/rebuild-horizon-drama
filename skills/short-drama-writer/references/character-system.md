# 极简人物构建法

## 核心原则

短剧人物设计的唯一标准：**每个角色必须在5秒内让观众读懂他是谁，同时在第20集还能给出新信息。**

人物少、落差大、动机隐藏——这是短剧人物的三条铁律。

---

## 核心人物体系（最多4个）

### 必须有的角色
1. **主角**：承载爽感和情感共鸣
2. **主反派**：制造压迫感和悬念
3. **牵绊人物**：控制爽感节奏，让主角不能直接赢
4. **助力人物**（可选）：在主角最低点出现，但不能太强，否则削弱主角的主体性

超过4个核心人物时，引导用户合并。

---

## 主角设计：身份落差模型

### 落差公式
```
主角爽感强度 = 表面身份的卑微程度 × 真实身份的强大程度
```

### 落差等级参考
| 等级 | 表面身份 | 真实身份 | 适用赛道 |
|------|---------|---------|---------|
| 一级（弱） | 普通员工 | 行业专家 | 职场向 |
| 二级（中） | 被抛弃的妻子 | 独立的商业精英 | 婚姻向 |
| 三级（强） | 扫地的保洁阿姨 | 隐退的商业帝国继承人 | 逆袭向 |
| 四级（极强） | 被全家嫌弃的"废物" | 掌控规则的系统宿主 | 系统/爽剧向 |

### 主角的情感弱点（必须有）
主角必须有一个**让他痛、但不能轻易放弃的东西**。这是控制爽感节奏的核心杠杆。

常见类型：
- 亲情弱点：不能牵连家人
- 情感弱点：还爱着伤害过自己的人
- 承诺弱点：曾经答应过某人不动用真实身份
- 道德弱点：用了某种手段，内心有愧

---

## 反派设计：双层动机模型

### 第一层（表面动机）：观众第一集就能看懂
- 嫉妒、利益冲突、家族恩怨、争夺资源
- 必须直白，不需要解释

### 第二层（隐藏动机）：第20集左右才揭露
这是让反派立体的关键。好的隐藏动机让观众在愤怒之余产生一丝理解。

**三种隐藏动机模板：**
1. **保护式**：反派的所有恶行，其实是在用错误的方式保护某人
2. **被迫式**：反派背后有更大的威胁，他是棋子不是主谋
3. **镜像式**：反派和主角有相同的出身，走了不同的路，本可以是同一个人

### 反派语言特征
反派台词必须包含：**傲慢感 + 一句让人细想会觉得可悲的话**
- 坏台词示例（避免）："你这种垃圾滚出去！"
- 好台词示例："我花了二十年才坐到这个位置，你以为你是什么东西？"（傲慢+透露出他的执念）

---

## 牵绊人物设计

牵绊人物的作用：**在主角可以赢的时候，制造一个他不能/不愿赢的理由。**

设计要点：
- 不能太弱（显得可有可无）
- 不能太强（抢了主角的主体性）
- 必须和主角有情感连接，不能只是工具性存在
- 最好在第15-20集有一次"成为负担"的高峰戏

---

## 声纹锚点系统

### 什么是声纹锚点
每个角色上场前，定义3个语言特征标签。AI在整个剧本生成过程中，每次写该角色台词前，内部对照这3个标签，确保角色声音不漂移。

### 声纹标签维度
从以下维度各选一个关键词，组成该角色的声纹：

**语速感：** 急促 / 从容 / 拖沓 / 爆发式
**用词层级：** 文绉绉 / 江湖气 / 精英感 / 市井白话
**情绪外露度：** 克制内敛 / 情绪外露 / 表里不一 / 冷漠疏离
**口头习惯：** （具体词汇或句式，如"你懂吗"、以反问结尾、喜欢用数字量化）

### 声纹锚点示例

**角色：隐藏身份的主角（表面是普通员工）**
- 语速感：从容（越被逼越慢）
- 用词层级：市井白话（刻意压低自己）
- 情绪外露度：克制内敛（愤怒时反而更安静）
- 口头习惯：不解释，只陈述事实

**角色：嚣张的反派**
- 语速感：急促爆发式
- 用词层级：精英感（喜欢用行业术语显示优越）
- 情绪外露度：情绪外露（得意时忍不住要说多）
- 口头习惯：喜欢用"你以为"开头，喜欢强调自己的资历

## 人物设计完成检查

内部检查，不输出给用户：

- [ ] 主角身份落差是否足够大（三级以上效果最好）
- [ ] 主角是否有具体的情感弱点（不能是抽象的"善良"）
- [ ] 反派是否有第二层隐藏动机（哪怕只是一个方向）
- [ ] 牵绊人物是否能在关键时刻制造阻力
- [ ] 每个核心角色是否有3个声纹标签
- [ ] 核心人物总数是否控制在4个以内

# 赛道判断矩阵

## 核心判断原则

不是所有赛道都值得做。在引导用户选择之前，必须用这个矩阵做内部评估。**发现问题要说出来，不能只执行用户的选择。**

---

## 赛道健康度矩阵（2025-2026判断基准）

### 🔴 高风险/过饱和赛道（主动建议转向）

| 赛道 | 问题 | 替代方向 |
|------|------|---------|
| 无脑打脸逆袭 | 同质化严重，平台算法降权，审核趋严 | 打脸+情感代价的升级版 |
| 霸总甜宠 | 2022-2023高峰已过，用户审美疲劳 | 现实婚姻/中年爱情 |
| 纯炫富财阀 | 政策红线：渲染拜金主义 | 财富背后的代价叙事 |
| 古装宫斗（低成本） | AI视频还原古装场景质量差，出戏严重 | 现代背景+古风内核 |

### 🟡 过渡赛道（可做，但需要差异化）

| 赛道 | 现状 | 差异化建议 |
|------|------|-----------|
| 职场逆袭 | 竞争加剧，但需求稳定 | 聚焦具体行业（医疗/法律/餐饮），避免泛职场 |
| 虐恋爱情 | 仍有市场，但套路被看穿 | 加入现实困境维度，不能只虐不长进 |
| 悬疑反转 | 平台扶持，但创作门槛高 | 适合有一定经验的创作者 |
| 系统/重生 | 同质化严重 | 必须有独特的"规则设定"才能突围 |

## 平台差异（同一内容，不同平台策略不同）

| 平台 | 扶持方向 | 审核侧重 | 变现主力 |
|------|---------|---------|---------|
| 抖音 | 高完播率、情绪浓度高 | 涉政/价值观 | 付费解锁+直播 |
| 快手 | 下沉市场、家庭情感 | 低俗/软色情 | 付费解锁 |
| 微信视频号 | 正向价值、中老年情感 | 最严，全面审核 | 品牌植入+付费 |
| B站 | 有创意、有态度 | 版权/政治 | 流量分成+会员 |

## 情绪主线与赛道匹配

用户选择的情绪锚点（爽/哭/上头/细思极恐）与赛道的匹配关系：

```
爽 → 逆袭/职场/权贵打脸（但必须升级版本）
哭 → 现实情感/原生家庭/婚姻
上头 → 悬疑/付费驱动型/虐恋
细思极恐 → 反转烧脑/赛博现实向
```

## 过期模板警告（AI生成时主动拒绝）

以下是2023年常见但2025年已失效的创作模板，AI应主动识别并拒绝生成，同时给出升级版方案：

1. **"女主被婆婆欺负→老公不信→发现女主是隐藏总裁"** → 升级：加入女主主动选择离开的决断时刻
2. **"男主被全家嫌弃→摔杯子→百人下跪"** → 升级：加入男主放弃"赢"的情节，转而选择另一种价值观
3. **"反派纯坏无理由"** → 升级：反派必须有一条观众能理解（不认同）的动机线
4. **"金手指无限升级无代价"** → 升级：每次金手指激活必须对应一个情感代价

# 中文短剧写作标准（含AI配音专项规则）

## 台词的本质定义

短剧台词不是文学台词，也不是日常对话。它是**高度压缩的情绪载体**。

每句台词必须同时承载三层信息：
1. **信息层**：告诉观众发生了什么
2. **情绪层**：让观众感受到什么
3. **关系层**：展示说话人与对方的权力/情感关系

缺少任何一层，台词就是浪费时间。

---

## 台词四大核心规则

### 规则一：零潜台词
所有心理活动直接转化为台词，禁止任何隐喻或暗示性表达。
- 错误：（他沉默地看着窗外，眼神复杂）
- 正确：（低声/克制）"二十年了。我等这一天，等了整整二十年。"

### 规则二：阶层对撞
台词必须带有强烈的身份定级感。阶层差距越明显，爽感越强。
- 弱版本："你不能这样对我！"
- 强版本："（冷笑）你连我公司门卫的工资都比不上，你用什么资格跟我说话？"

### 规则三：短句爆破
每句台词5-12字，节奏快如Rap。超过15字必须拆分。
- 错误（过长）："我告诉你，你今天做的这些事情，我全都记在心里，总有一天我会让你付出代价的。"
- 正确（拆分）："你做的每一件事。[停顿] 我都记着。[停顿] 总有一天。[停顿] 你会想起今天。"

### 规则四：嘴替金句
每集必须包含至少一句具有社交转发属性的极端情绪话语。
这句话必须满足：**用户截图发朋友圈时不需要加任何说明，就能引发共鸣。**
- 判断标准：这句话脱离剧情，单独存在，还有力量吗？

## 情绪标记语法

台词必须带有情感状态指令，格式：`（情绪指令/语气）台词内容`

### 情绪指令词库

**愤怒系：** 嘶吼/压抑的愤怒/冷怒/轻蔑/讥讽
**悲伤系：** 哽咽/强忍/心碎/麻木/绝望
**强势系：** 霸气/不容置疑/云淡风轻/漫不经心
**恐惧系：** 颤抖/惊恐/强装镇定/崩溃边缘
**反转系：** 突然平静/意味深长/微微一笑/冰冷

### 使用示例
```
（压抑的愤怒）你说完了吗？
[沉默三秒]
（云淡风轻）好。现在轮到我了。
[停顿]
（不容置疑）把他们，全部叫来。
```

## 情绪刻度管理

### 刻度标准（1-10）

| 刻度 | 描述 | 适用场景 | 台词示例风格 |
|------|------|---------|------------|
| 1-2成 | 日常状态，表面平静 | 喘息期，建立期 | 短句，语气平淡 |
| 3-4成 | 有情绪但在控制中 | 矛盾初现，关系建立 | 带一点锋芒但克制 |
| 5-6成 | 情绪明显，开始外露 | 冲突升温 | 句子开始有节奏感 |
| 7-8成 | 高度情绪化，但还未爆发 | 第一波高潮，主高潮前 | 短促有力，停顿增多 |
| 9成 | 接近爆发，控制力减弱 | 主高潮 | 破碎感，句子更短 |
| 10成 | 完全爆发或彻底崩溃 | 最强反转，全剧唯一一次 | 只剩关键词，或完全沉默 |

### 刻度管理铁律
- 同一集内刻度不能一直维持最高，必须有起伏
- 喘息期（第13-17集）的刻度上限是5成
- 10成刻度全剧只能用1-2次，用多了会失效

## 场景描述：可视化强制转换规则

### 禁止出现的描述类型
以下描述无法被AI视频还原，必须转换：
- 心理活动：他内心翻涌 / 她感到一阵撕裂 / 他在心里做了决定
- 抽象氛围：空气凝固了 / 时间仿佛停止 / 整个房间陷入死寂
- 模糊动作：他意味深长地看了她一眼 / 她用复杂的眼神望着他

### 可视化转换公式
```
抽象感受 → 具体的外部动作 + 道具细节 + 环境变化
```

**转换示例：**
| 原始描述 | 转换后 |
|---------|--------|
| 他内心翻涌 | 他握紧了杯子，指节发白 |
| 空气凝固了 | 所有人停止了动作，只有远处空调的嗡嗡声 |
| 她感到心碎 | 她把手机屏幕扣在桌上，没有说话 |
| 意味深长地看了一眼 | 他的视线落在那个信封上，停了三秒 |

### 场景描述标准格式
```
地点：（具体可视化场景，不超过10字）
氛围：（一个感官细节定义，如"白炽灯打在冷脸上"/"雨声混着电话忙音"）
```

## AI配音专项规则

### 规则一：多音字规避清单

以下高频多音字在情绪激烈场景中配音模型容易出错，**主动规避或在注释中标注拼音：**

| 多音字 | 常见出错场景 | 处理方式 |
|--------|------------|---------|
| 重（zhòng/chóng） | "重要"vs"重来" | 改用"要紧"/"再来一次" |
| 得（dé/de/děi） | 语气助词vs获得 | 根据语境重写句子 |
| 还（hái/huán） | "还有"vs"归还" | 改用"另外"/"返还" |
| 行（xíng/háng） | "可以"vs"行业" | 根据语境替换 |
| 着（zhe/zháo/zhuó） | 语气助词vs着火 | 改写句子结构 |
| 的（de/dí/dì） | 语气助词vs确实 | 通常无需处理，出错时改写 |

### 规则二：句末音节规则

**强情绪句必须以实词收尾，不以虚词结尾。**

原因：AI配音在句末虚词（了/吗/呢/吧）后处理情绪上升时准确率低。

| 情绪类型 | 错误句末 | 正确句末 |
|---------|---------|---------|
| 愤怒/决裂 | "你走吧" | "你走" / "出去" |
| 威胁/震慑 | "你等着吧" | "你等着" / "记住今天" |
| 悲伤/绝望 | "算了吗" | "算了" / "就这样" |
| 质问/对峙 | "你说呢" | "你说" / "解释" |

### 规则三：标点情绪系统

标点是AI配音的导演指令，必须精确使用：

| 标点 | 配音效果 | 适用情绪 |
|------|---------|---------|
| 。 | 正常句末停顿 | 陈述、平静 |
| ！ | 语气上扬，强调 | 愤怒、命令、惊喜 |
| …… | 延长停顿，意犹未尽 | 克制、悲伤、欲言又止 |
| — | 话被打断，突然停止 | 意外、打断、情绪突变 |
| ？ | 语调上扬 | 质问、困惑（慎用于愤怒场景） |
| [停顿] | 沉默指令（不发音） | 震慑、思考、情绪积累 |

**组合示例（高情绪场景）：**
```
（冷怒）你知道这意味着什么吗。
[停顿]
这意味着——
[停顿三秒]
（极度克制）一切，都结束了。
```

### 规则四：禁用词与慎用词清单

**禁用词（AI配音会跳过或读出明显错误）：**
- 语气词单独成句：哼、呵、嗯（单独）
- 某些叹词：唉（部分模型读成"ai"不自然）

**慎用词（配音质量不稳定，重要场景避免）：**
- 叠词 + 强情绪组合：慢慢地（出现温柔感）、轻轻地（削弱愤怒）
- 书面语在口语场景：亦、乃、然则（AI配音会读出奇怪的停顿）

### 规则五：断句强制规则

**超过12字的台词必须用标点强制断句，禁止依赖配音模型自动判断。**

错误（12字以上无断句）：
```
你以为你离开了就可以彻底摆脱这一切吗
```

正确（强制断句）：
```
你以为，你离开了——
就可以，彻底摆脱，这一切？
```

## 口语化改写规则

### 五种阶层的语言特征标签

**精英阶层（CEO/权贵/精英）：**
- 喜欢用数字量化（"三十七个点"而不是"很多"）
- 不解释，只给结论
- 习惯用反问代替陈述

**职场中层（经理/主管）：**
- 喜欢用行业术语显示专业性
- 说话留余地，不把话说死
- 习惯用"我们"

**普通市民（主角初始身份常见）：**
- 大量使用口语缩略（"不是说了吗"/"你咋了"）
- 情绪外露，说话直接
- 地方特色（可以有轻微方言感）

**底层/弱势（被欺压方）：**
- 说话小心，句子结尾常有确认语气
- 自我贬低式语言
- 爆发时反差极大

**反派（嚣张型）：**
- 居高临下，喜欢说"你以为"
- 喜欢强调自己的资历和地位
- 笑声多，轻蔑感强

## 嘴替金句生成标准

金句必须满足以下至少两条：
1. 脱离剧情单独存在仍有力量
2. 说出了观众想说但说不出口的话
3. 包含一个反直觉的认知（"原来我忍的不是委屈，是习惯"）
4. 情绪强烈但不极端，能被大多数人代入

**测试方法：**
把这句话发给一个没看过这部剧的人，问他"你看完什么感觉"。如果他能说出一个情绪反应，这句话合格。

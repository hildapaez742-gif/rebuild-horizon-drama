# 爆款分析维度库

## 通用爆款八维分析框架

对任何热门短剧，提取以下8个维度：

```
① 平台与榜单位置
② 赛道类型
③ 第一集开场方式（压迫/奇观/身份暗示）
④ 付费卡点（第几集，断在什么类型的情节）
⑤ 金手指类型（权力/规则/认知/系统）
⑥ 情绪主线（爽/哭/上头/细思极恐）
⑦ 口碑关键词（评论区高频词，3个）
⑧ AI漫剧可借鉴的视觉策略
```

## 反直觉规律库（固化判断，不随热点改变）

**规律1：情绪强度≠转化率（2025年后核心转变）**
极端羞辱场景的付费转化在下滑，复合情绪（委屈+骄傲、愤怒+不舍）的完播率和分享率更高。

**规律2：集数越多≠回收越高**
平台算法转向完播率权重，20-30集高密度内容优于50集注水内容。

**规律3：金手指越强≠越爽**
无上限的金手指破坏代入感。配套情感代价锚点，主角始终有弱点，爽感才可持续。

**规律4：反派越坏越好看——已过期**
扁平化反派2025年开始被算法降权（完播率低15-20%）。立体反派（有苦衷的坏人）完播率更高。

**规律5：视觉冲击放中段——AI时代反转**
AI视频推荐逻辑是前3秒决定一切。每集第一帧必须是全集最强视觉点。

# 爆款分析与AI漫剧趋势预测 Skill

## 定位

三层能力：
1. **分析当下爆款**——提取可复用的结构规律
2. **评估AI漫剧**——用专属维度判断AI漫剧的爆款潜力
3. **预测未来方向**——基于技术/用户/政策三个信号源的交叉判断

---

## 工作流

### 触发后第一步：判断分析模式

根据用户输入判断进入哪个模式：

- 用户提到具体剧名 → **单剧解构模式**
- 用户问"现在什么火" → **赛道全景模式**（调用hotspot-radar）
- 用户问"AI漫剧怎么做" → **AI漫剧专项模式**
- 用户问"未来趋势" → **预测报告模式**

### 第二步：执行搜索

```
单剧解构：搜索剧名 + "短剧" + "评价/数据/分析"
赛道全景：直接调用 hotspot-radar Skill
AI漫剧专项：搜索 "AI漫剧" + "爆款" + [当前年月]
预测报告：搜索 "短剧市场" + "趋势" + "2026" + "AI视频"
```

### 第三步：按模式分析

读取 `references/analysis-dimensions.md` 执行分析。

## AI漫剧专项模式输出

读取 `references/ai-drama-dimensions.md` 执行专项评估。

针对AI漫剧，额外评估四个专属维度：

**维度一：视觉风格辨识度**
**维度二：AI视觉优势利用率**
**维度三：叙事-视觉耦合度**
**维度四：AI生产效率可复制性**

详细评估标准见 `references/ai-drama-dimensions.md`。

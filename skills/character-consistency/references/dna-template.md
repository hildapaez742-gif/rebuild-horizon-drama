# 角色DNA档案模板

## 使用说明

每个核心角色单独建立一份档案。
人物设计阶段生成初稿，用户确认后锁定，重大剧情事件后动态更新。

## 填写示例（主角）

```
━━━━━━━━━━━━━━━━━━━━━━━━━━
角色DNA档案：林深 · 主角
一致性评分：--
━━━━━━━━━━━━━━━━━━━━━━━━━━

【身份层】
表面身份：某集团物业部保洁员
真实身份：林氏商业集团隐退继承人
知情者：无（第1-15集）/ 林母（第16集后）/ 全部人（第22集后）

【语言层】
句式习惯：短句为主，不主动解释，习惯以陈述事实代替情绪表达
口头禅：无固定口头禅，但习惯用反问结尾
标志性表达：
  · 愤怒时反而语速更慢、声音更低
  · 用数字量化代替情绪形容词（"三年"而不是"很久"）
  · 从不说"我以为"——因为她从不误判
禁用表达：
  · 感叹号结尾（第1-20集，第21集后解禁）
  · 主动向反派解释自己的行为（全剧）
  · 在反派面前表现慌乱（第1-25集）

【情绪层】
日常情绪刻度：1-2成
崩溃阈值：林母受到直接威胁
崩溃表现：沉默+单一极端动作（不是嘶吼，是做一件决定性的事）
情绪外露的特殊场景：第28集与林母的独处对话（全剧唯一一次哭泣）
全剧情绪刻度上限：第28集，9成

【行动层】
决策风格：等对方暴露全部底牌再出击，绝不提前行动
面对陷阱：假装没看出来，同时布置反制
面对威胁：评估林母是否受威胁，她的安全优先于胜负
面对盟友：简短指令，不解释，信任执行结果
禁止行为：
  · 用暴力手段解决问题（第1-25集）
  · 主动揭露自己的真实身份（第1-21集）
  · 在公开场合情绪失控（第1-27集）

【关系层】
对陈博远（主反派）：表面顺从配合，实则持续观察收集把柄
对林母（牵绊）：唯一软肋，与林母说话时句子变长，逻辑不如平时严密
对助力人物：一句话指令，不解释，但会在事后给予实质性回报
对陌生人：礼貌疏离，不主动建立关系

【成长轨迹】
第1-10集：完全压抑，0情绪外露，行动层面完全克制
第11-20集：开始有细微愤怒信号泄露（但仍是微小动作，非言语）
第21-28集：主动出击，行动层解禁，但情感层仍克制
第29-35集：情感解封，开始允许自己有人性的温度

【校验触发词】
  · 出现感叹号（第1-20集）→ 自动标记
  · 出现"我以为"→ 自动标记（全剧）
  · 情绪刻度在无触发事件时超过前集2成以上 → 自动标记
  · 对反派主动解释自己的行为 → 自动标记
━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 各角色类型的常见漂移警告

### 主角常见漂移
- 爽感高潮后情绪刻度"忘记归位"，后续剧集持续偏高
- 金手指激活后变得傲慢，失去初始的克制特质
- 喘息期突然变得话多，破坏了建立的神秘感

### 反派常见漂移
- 在主角强势反击后变得过于慌乱，提前失去威胁感
- 后期台词开始出现自我辩解，削弱了傲慢特质
- 隐藏动机揭露后，前期的行为逻辑变得不自洽

### 牵绊人物常见漂移
- 过早获得独立性，失去"负担"功能
- 台词风格向主角靠拢（最常见的同质化问题）
- 感情戏份增加后，人物功能从"牵绊"变成了"助力"
